# 📃 Documentación

## Sobrescribir Configuración

A través de las siguientes propiedades en el YAML, se puede sobrescribir la configuración para mostrar o ocultar caracteres deseados^[No los describo debido a que es algo obvio lo que hacen]:

- `cc` (Control Characters): Mostrar o ocultar los caracteres en la nota
- `cc-selection`
- `cc-tab`
- `cc-newline`
- `cc-space`

# ⚙️ Configuración

- Only show control characters in selection: `TRUE`

## Show

- Space: `TRUE` *(Default)*
- Tab: `TRUE` *(Default)*
- New Line: `TRUE` *(Default)*

	- Dark: *(Default)*

- Symbol for other non printing control characters (Icon SVG URI encoded): *(Default)*
